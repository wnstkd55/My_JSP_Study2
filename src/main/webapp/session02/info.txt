JSP의 내장 객체(7가지) : Import를 사용하지 않고 바로 사용할 수 있는 객체
	*.jsp : <% 스크립트릿태그%>
	
	1. request
	2. response
	3. application
	4. session
	5. out
	6. exception
	7. Cookie
	
	
	객체의 사용 범위
	1. page 영역 : 동일한 페이지 영역에서만 공유, 페이지를 벗어나면 소멸됨
	2. request 영역 : 클라이언트의 요청에 의해서 호출된 페이지와 포워드(요청 전달)된 페이지 까지 공유됨.
			새로운 페이지를 호출시 소멸됨
	3. session 영역 : 클라이언트가 처음 접속한 웹 브라우저를 닫을때까지 공유됨.
		포워드나 새로운 페이지 이동 후에도 소멸되지 않는다.
		소멸 시기 : 브라우저 닫을때, 서버의 서버의 세션 유효시간이 지나면 소멸
		해당 세션 ID에서만 공유, 즉 다른 사용자와는 공유되지 않는다.
	4. Application영역 : 웹 Application이 종료될때까지 유지됨.
			소멸 시기 : 웹 서버가 종료 될때까지 남아있음.
